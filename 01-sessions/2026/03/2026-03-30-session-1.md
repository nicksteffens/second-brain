### Most Valuable Collaboration
Deep research on the miragejs ecosystem — understanding that miragejs-orm is purely an ORM with no request interception, that mirage-msw and mirage-pretender only work with the full MirageJS server, and that Mirage is only used in Cypress tests (never dev mode) — which means cy.intercept() fully replaces Pretender with zero new dependencies.

### Key Insight
Mirage's request interception (Pretender) is only used in Cypress tests — dev mode uses Vite proxy to Rails. This means cy.intercept() can fully replace Pretender without MSW, making the migration path: miragejs-orm (typed ORM) + cy.intercept() (Cypress-native interception) = drop miragejs entirely and remove 210 KB production chunk.
