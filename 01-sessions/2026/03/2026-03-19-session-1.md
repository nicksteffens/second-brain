### Challenges Encountered
- CI validate workflow broke on multi-file PRs — GITHUB_OUTPUT can't handle newlines with key=value syntax, then bash tried to execute filenames as commands. Fixed with `tr '\n' ' '`
- Release race condition — generate-index auto-commit on main collided with release workflow's version bump push. Fixed by moving index generation to PR trigger
- Plugin update bootstrapping — update command only available in v1.8.0+, users on older versions can't use it to update. Documented manual upgrade path
- Marketplace index staleness — `claude plugin update` reports "already at latest" without first refreshing marketplace. Added marketplace update step

### Most Valuable Collaboration
Plugin ecosystem work — building the update command, issue templates, CI validation, and auto-generated index transformed the project from a personality collection into a proper maintained plugin with contributor infrastructure.

### Key Insight
Squash merge destroys conventional commit subjects, which is why the UI repo's changelog is empty. The release pipeline config is correct — the input is garbage because squash collapses all commits into a PR title that doesn't follow conventional commit format. Filed issue #629 on movableink/ui.
