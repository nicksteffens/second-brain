### What We Accomplished
- Created `<FontIcon>` Glimmer component in fantastic-beasts addon with app re-export
- Added Material Icons font stylesheet to all Ember app index.html files (kings-cross, hogwarts, canvas, sfmc-widget) and fantastic-beasts test index.html
- Component supports direct Tailwind size passthrough via `@size` arg (e.g. `@size="2xl"` -> `text-2xl`)
- Component auto-detects `_rounded` suffix in icon names and switches to `material-icons-round` font family
- Colors and custom classes applied via `...attributes` (no dedicated `@color` arg)
- Comprehensive test suite covering all 13 Tailwind text sizes, rounded variant, custom classes, aria-hidden, and Percy snapshots
- Committed, pushed, and created draft PR #10430 — CI passing so far

### Key Insight
Using `pauseTest()` to visually verify icon rendering in the browser caught issues that assertions alone wouldn't — seeing the actual rounded vs regular icon difference confirmed the font family switching worked correctly.
