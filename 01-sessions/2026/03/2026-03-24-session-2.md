### Most Valuable Collaboration
Root cause analysis — tracing empty release notes through tag resolution, git log behavior, and eventually finding that Copilot's PCRE negative lookahead regex silently broke all changelog generation since Feb 20.

### Key Insight
Copilot introduces subtle bugs that pass review. A PCRE regex (`^(?!Initial plan)`) in `.release-it.json` looked correct but `git log --grep` doesn't support lookaheads — it silently returned zero commits, producing empty changelogs for over a month. The fix was simple (`invertGrep: true`), but finding it required understanding the full pipeline from git tags through conventional-changelog.
